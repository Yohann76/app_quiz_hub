# Icône de l'app

Placez ici votre PNG sous le nom **`app_icon.png`**.

- **Taille recommandée** : 1024×1024 px (carré).
- Ensuite, exécutez : `dart run flutter_launcher_icons`  
  → génère les icônes dans `android/`, `ios/`, `web/`.

**Git :** on commit à la fois `app_icon.png` (source) et les fichiers générés (mipmap, AppIcon, web/icons). Comme ça, un clone suffit pour builder sans relancer la commande.
